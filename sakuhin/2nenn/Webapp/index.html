アンティポブ・野中・渡辺

使用したツール

ーー作品のこだわりポイント！！

ランチをなかなか決めることができないという悩みが多くあったため、その悩みを解消するためのアプリを制作した点です。

ーーこの制作で挑戦したこと

Typescript を使用したため、通常の Javascript を使用する場合よりも困難でしたが、このテクノロジーのおかげで、アプリケーションのデバッグははるかに簡単になりました。 また、アプリケーションの機能を改善するための適切な作業モジュールを見つけるのが困難でした。多くの情報は古く、React-nativeの現在のバージョンには適していませんでした。 モジュールのソースを探して手動で変更し、デバッグしてアプリケーションに合わせて調整する必要がありました。 また、モバイル オペレーティング システムのすべてのバージョンのバグを修正することも非常に困難でした。 Android、iOS、android emulator - それらはすべて独自の機微を持っているため、アプリケーションが突然正しく動作しなくなったり、完全に動作しなくなったりする可能性があったため大変でした。

ーー苦労したところ

アプリケーションをゼロから構築することです。どこから始めて、必要な機能をどのように実装するかを理解するのがとても困難でした。 ただし、1 つの大きな問題を小さなブロックに分割し、それぞれを個別に考えれば、解決策を自然に得ることができました。
